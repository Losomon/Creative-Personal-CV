## About

This repository contains the frontend code for Solomon Mwangi's personal CV portfolio — a full-stack developer & UI enthusiast brand site. The design uses a dark, gold-accented theme with smooth scroll animations, stat highlights, and a strong visual hierarchy to present a professional, hire-ready presence.

## ✨ Features

- **Hero section** with animated headline, key stats (projects completed, years of experience, client satisfaction), CTA buttons (*View My Work*, *Hire Me*), and an availability badge
- **My Approach** — four-step process cards (Design First, Clean Code, User Focused, Performance Driven)
- **About Me** — personal intro, location, language, availability, and a tech-stack tag cloud
- **Core Expertise** — skill category cards (Frontend, Backend, Mobile, Database, UI/UX, DevOps & Cloud) with progress bars and tech tags
- **Portfolio** — filterable project grid with a featured project showcase (live demo + source code links) and smaller project cards
- **Experience & Milestones** — year-by-year timeline of career progress
- **Client Testimonials** — star-rated quotes from past clients
- **Contact** — contact form, direct email/social links, and response-time note
- Smooth scrolling navigation and responsive layout across devices

## 🛠️ Tech Stack

- **React 18**
- **Vite 5**
- JavaScript (ESM)
- CSS modules / styles

## 🚀 Getting Started

### Install dependencies

```bash
cd frontend
npm install
```

### Run locally

```bash
npm run dev
```

Open the local URL shown in the terminal to view the portfolio site in your browser.

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
